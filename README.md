## 개발 및 실행환경

- CentOS 8.2
- Python 3.6.8
- Flask 1.1.2
- MySql 8.0.17

## 사용 라이브러리

1. JWT - 사용자 인증토큰 발행에 사용
2. Bcrypt - 비밀번호 단방향 암호화에 사용
3. Jsonschema - 사용자 입력값 유효성 검증을 위해 사용
4. Pymysql - MySql DB연동을 위해 사용
